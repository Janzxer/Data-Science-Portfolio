# Data Science & Analytics Portfolio

A collection of Machine Learning projects demonstrating Supervised and Unsupervised learning techniques using Python (Scikit-Learn).

## Project Index

### 1. [Predictive Fuel Efficiency Model](./01_AutoMPG_Regression.ipynb)
* **Type:** Supervised Learning (Regression)
* **Goal:** Predict vehicle MPG based on engine specifications.
* **Technique:** Linear Regression with algorithmic feature selection (Itertools).
* **Result:** Successfully modeled non-linear relationships using feature engineering.

### 2. [Wine Quality Classification](./02_Wine_Classification.ipynb)
* **Type:** Supervised Learning (Classification)
* **Goal:** Categorize wine cultivars based on chemical analysis.
* **Models:** K-Nearest Neighbors (KNN) and Support Vector Classifier (SVC).
* **Validation:** Evaluated using Confusion Matrices and Accuracy Scores.

### 3. [Life Expectancy Clustering](./03_LifeExpectancy_Clustering.ipynb)
* **Type:** Unsupervised Learning (Clustering)
* **Goal:** Group countries by demographic and health statistics.
* **Technique:** K-Means Clustering & Principal Component Analysis (PCA).
* **Visuals:** 2D visualization of high-dimensional data clusters.

### 4. [Workplace Satisfaction Analysis](./04_Workplace_Survey_Analysis.ipynb)
* **Type:** Statistical Analysis
* **Goal:** Identify correlations between employee age, salary, and happiness.
* **Technique:** Pearson Chi-Square contingency tests.

## Tech Stack
* **Languages:** Python 3.x
* **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn
